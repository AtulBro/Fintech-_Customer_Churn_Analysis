# 📊 FinTech Customer Churn Analysis Dashboard | Power BI

## 📌 Project Overview

The **FinTech Customer Churn Analysis Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI**. The dashboard helps organizations analyze customer behavior, identify churn patterns, monitor key business metrics, and support data-driven decision-making through dynamic KPIs and interactive visualizations.

The primary objective of this project is to provide actionable insights that enable businesses to improve customer retention, reduce churn, and optimize overall customer engagement.

---

# 🚀 Features

## 1️⃣ Executive Dashboard

The Executive Dashboard provides a high-level overview of customer performance and key business metrics.

### 📈 Key Performance Indicators (KPIs)

- 👥 Total Customers
- 📉 Churn Rate (%)
- ✅ Active Customers
- 💰 Total Account Balance
- 💳 Total Transaction Amount
- 📊 Average Credit Score

### 📊 Visualizations

- 🌍 Balance by Geography
- ⚠️ Customers by Risk Category
- 📅 Monthly Churn Trend
- 🏷️ Transaction Amount by Customer Segment

The dashboard also includes interactive slicers that allow users to filter data by:

- Geography
- Gender
- Customer Segment

---

## 2️⃣ Customer Insights Dashboard

The Customer Insights Dashboard focuses on customer behavior, risk analysis, and retention trends through interactive visualizations.

### 📊 Visualizations

- 📍 Churn Rate by Geography
- 👤 Active vs Inactive Customers
- 🛡️ Customers by Risk Category
- 💳 Loan Status Analysis
- 💼 Customer Segment Analysis
- 📈 Credit Score & Balance Analysis
- 🔍 Interactive Filters & Drill-down Analysis

This dashboard helps businesses identify high-risk customers, analyze customer segments, and make strategic decisions to improve retention.
